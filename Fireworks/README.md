[在线演示](https://0-v-0.github.io/Fireworks/)

## 特点
- 每隔一定时间自动放烟花
- 双击或右键切换自动放烟花频率
- 鼠标点击哪就在哪放烟花，支持拖动
- 默认隐藏光标
- 建议全屏模式玩耍
